Lab 1: My Favorite Websites

About This Page

This page was built as part of Lab 1 to practice basic HTML — headings, paragraphs, lists, and links. I decided to make it about websites I genuinely visit often rather than just picking random ones.

My Top Three Favorite Websites

  1. YouTube

    I spend a lot of time on YouTube, mostly watching tutorials, tech reviews, and the occasional documentary. It is honestly one of the best places to learn almost anything for free.

  2. GitHub

    GitHub is where I explore open source projects and store my own code. It has been useful since I started learning programming — being able to see how other people write code is a big help.

  3. Wikipedia

    Wikipedia is my go-to whenever I need a quick explanation on something I don't fully understand. The depth of information on there for almost any topic is impressive.
